# 타입 종류
# * FEAT : 새로운 기능 추가
# * MODIFY : 기존 기능 수정
# * FIX : 버그 수정
# * DOCS : 문서 수정
# * STYLE : 코드 스타일 수정
# * REFACTOR : 코드 리펙토링
# * CHORE : 빌드 부분 혹은 패키지 매니저 수정사항

# 제목 규칙
# * 제목과 본문은 띄어쓰기로 구분함

# 본문 규칙
# * 본문의 각 행은 72글자로 제한
